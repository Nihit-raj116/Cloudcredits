# Simple Calculator

This is a basic command-line calculator written in Python.  
It can perform the following operations:
- Addition
- Subtraction
- Multiplication
- Division

## Features

✅ Simple and easy to use  
✅ Handles division by zero  
✅ User-friendly input prompts  

## Requirements

- Python 3.x
